Docker Images for Malware Analysis

The REMnux project provides Docker images of popular malware analysis tools, with the goal of allowing investigators to conveniently utilize difficult-to-install applications without having to install the REMnux distro.

This section is a work in progress.

Using and Building Docker Images

If you are new to Docker, you can learn how to distribute and run apps inside containers. You can see the images built to date on the REMnux Docker hub repository.
